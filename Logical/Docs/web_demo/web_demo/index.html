B&R Example Pages

This are the example pages for the B&R Web Server. Some pages are accessing PVs from the B&R Coffee Machine Demo. They may used on the ARsim of this project.

Notice: If you have open this page from the explorer, configure the web root directory of the ARsim to point directly to this directory and access this page via http://127.0.0.1 to see the working ASP Demos.

Interactive Pages with ASP Goform - Accessing PVs

Please note: ASP User are mentioned seperatly.

HTML Form text version Use web service Goform/ReadWrite to switch on the coffee machine
HTML Form (Multiform) text version Use web service Goform/ReadWrite to get the change for the coffee
.NET Application (Download Demo) source file (br_ASPGoform.cs) Use web service Goform/ReadWrite in a .NET application (internal usage: BR_ASP_Request.PV_Access.ReadPV("IP","PVName"); )
Java Application (Download Demo) source file (br_ASPGoform.java) Encapuslated usage of the web service Goform/ReadWrite in an Java application. Downloaded demo have to be started from command line (java -jar ASP_Access.jar)
Javascript (HTML) text version Encapuslated usage of the web service Goform/ReadWrite in HTML with Javascript
Javascript (SVG) text version Encapuslated usage of the web service Goform/ReadWrite in SVG with Javascript
source file (PV_Access.js) text version Javascript functions for encapuslated usage of Goform/ReadWrite
Helper File For .NET, Java And Javascript Demos text version Returns the value of the requested PV as a document
Active and interactive pages elements combined

ASP User: Login And Usage
Login text version Form one shows an example form. Form two is prefilled with a valid login for direct use.
Logout text version Logout of the ASP User
Enviroment Variables text version Use EnvPrint to print enviroment variables
ASP User Specific page content (Target Page For Login And Logout) text version Use ErrorMsg and UserGroup to control the processing of the ASP file. As ASP User "asp1:pw1" (group 1) the coffee machine can be switched on. As ASP User "asp2:pw2" (group 2) the coffee machine can be switched off.
Target Page For Failed Login text version User will be redirected to this page, if login of ASP User failed. This target page is configured as hidden field in the login form.
BrWebSvr.cfg - Configuration Of ASP Users - Configuration file for ASP Users used for this examples.
